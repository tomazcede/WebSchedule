# WebSchedule

WebSchedule je sistem za ustvarjanje in upravljanje personaliziranih urnikov.

## Funkcionalnosti

- Ustvarjanje in urejanje urnikov ter dogodkov  
- Objavljanje dogodkov (drugi uporabniki jih lahko dodajo v svoj urnik)  
- Registracija in prijava uporabnikov  
- Lokalno shranjevanje urnika za neprijavljene uporabnike (goste)
- Prikaz istega urnika na spletu ter v mobilni aplikaciji

## Tehnologije

- **Backend (API):** Laravel  
- **Spletni vmesnik:** Nuxt.js  
- **Mobilna aplikacija:** Nuxt.js + Capacitor
